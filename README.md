# Amit Sahu - Developer Portfolio

My personal portfolio website showcasing projects, skills, and experience as a full-stack developer.

## Live Demo

> Visit the live site: [amitsahu11-portfolio on GitHub Pages](https://amit-sahu11.github.io/amitsahu11-portfolio)
>
> ## Built With
>
> - HTML5 (semantic structure)
> - - CSS3 (animations, flexbox, grid)
>   - - JavaScript (interactivity)
>    
>     - ## Sections
>    
>     - - Hero - Introduction and call-to-action
>       - - About - Background, skills, and tools
>         - - Projects - Featured project cards with links
>           - - Contact - Social links and email
>            
>             - ## Running Locally

>               # Clone the repo

>
>               # Open in browser

>               ```
